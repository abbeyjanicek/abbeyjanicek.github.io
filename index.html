Solo Project: Superior Hiking Trail Planning Web App

    I created this app because I have an interest in hiking the Superior Hiking Trail in northern Minnesota and, while the associated website has a lot of information, I found it to be overwhelming. This app is my way of simplifying some of the information available.
    A registered user is able to plan a future hike, record previous hikes, and rate and review campsites along the trail.
    Technologies: Angular JS, React, Redux, Postgresql, CSS, Material UI
